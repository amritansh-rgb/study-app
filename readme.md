# StudySprint

StudySprint is a simple study helper web app that runs fully in the browser.

## Features

- Add and complete study tasks
- Track priority by subject
- Use a Pomodoro focus timer
- Save revision notes automatically
- Create flashcards for quick recall
- Keep progress saved in local storage
